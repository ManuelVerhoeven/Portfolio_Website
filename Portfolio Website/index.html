Peachful Gnome

This is a personal project in which I challenged myself by creating a 3D scene containing a character model in an environment with an accompanying animation. The challenge for this project was to build on my maya skill set and to create a high quality piece of media with the bonus objective to make it bright and happy. My project solution was a Gnome character sitting on the edge of a mushroom in a grass clearing playing a flute. The skills that I wanted to further develop with this project was my 3D modeling and animation skills in maya, I wanted a project that would allowed me to use a diverse range of tools, this project taught me a lot through problem solving.

Educational Animation on the Stomach

In this project the brief required me to create an educational animation on the stomach that teaches the viewer the stomachs basic functionality. My project solution was to create a 3D model of the digestive system and use 2D animation on top of that 3D model to show the inner workings of the stomach. I was interested in making a 3D/2D hybrid animation, as I had not attempted it before, and it would be a challenge to make it work together. The style that I wanted to achieve in this animation is an old medical journal look that comes across as sophisticated.

The Missing Link

The aim of this project was to take an existing character model and create uv maps to give the character texture and colour, the other requirement was to provide an accompanying animation. My solution to this project was to create the missing link in human evolution, I studied monkey fur and early human clothing for the most realist result. For the animation I looked at typical monkey behavior to create primitive movements.
